  - Java 11

## 4. 핵심 기능
**`velo9`는 단순하고 직관적인 사용이 가능한 `웹 기반 블로그 서비스`입니다.** <br><br>
velo9는 블로그 활동에 필요한 다양한 편의 기능을 제공합니다.<br>
누구나 쉽고 간편하게 `포스트 작성`이 가능하며, 태그와 시리즈 정보를 활용해 `포스트를 빠르게 탐색`할 수 있습니다.
